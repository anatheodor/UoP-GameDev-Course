Εδώ θα βρείτε όλο τον πηγαίο κώδικα (`.rpy` scripts), τα assets και τα παραδείγματα που τρέχουμε κατά τη διάρκεια των διαλέξεων πάνω στη μηχανή **Ren'Py**. ## 🛠️ Απαιτούμενα Εργαλεία

Για να τρέξετε και να επεξεργαστείτε τα παραδείγματα αυτού του repository, θα χρειαστείτε:

1. **Ren'Py Visual Novel Engine:** Κατεβάστε το τελευταίο SDK από το [επίσημο site (renpy.org)](https://www.renpy.org/latest.html).
2. **Git:** Για να κάνετε clone και pull τις τελευταίες αλλαγές (αν και λογικά το έχετε ήδη).
3. **Έναν Code Editor:** Προτείνεται το **Visual Studio Code** με το αντίστοιχο Ren'Py Extension για syntax highlighting, ή το Atom/Editra που προσφέρει το ίδιο το Ren'Py launcher.
